Pusat Informasi (FAQ)

1. Bagaimana mekanisme pendaftaran mitra baru?

Verifikasi Farm NKV -> Aktivasi Digital ID di sistem FreshEgg -> Akses Market Hub melalui WhatsApp API Fonnte.

2. Keamanan Data & Kepatuhan Regulasi (UU PDP)?

Data dilindungi AES-256 Encryption dan sepenuhnya patuh pada UU PDP No. 27/2022. Server monitoring bersifat restricted dan terlindungi firewall berlapis.
